# koaMovie

> Koa2 实现电影微信公众号前后端开发

## Tech Stack

- Node.js
- Koa2
- MongoDB
- mongoose
- 微信 JS-SDK
- Pug模板引擎
- Bootstrap
- HTML
- CSS
- JavaScript
- JavaScript Standard Style
- ES6 / ES7
 
## Build Setup

```bash
# install dependecies
npm install

# start app
npm run start

# check the style of all JavaScript files
npm run precommit
```
